Proverb Being

What is Proverb?

Proverb is a decentralised digital currency with near-instant transaction speeds and negligible transaction fees built upon Proof of Stake 3.0 (PoSV3, BPoS) as introduced by the Proverb under the development of AYCHDeveloper.

What is Proverb Being?

Proverb Being is the name of open source software which enables the use of this currency. It takes Proverb to the next level by building upon Bitcoin Core 0.13.2 with some patches from newer Bitcoin Core versions to offer performance enhancements, wider compatibility with third party services and a more advanced base.

How to upgrade?

If you are running Proverb Legacy wallet v1.2.x:

You need to download the blockchain and send your funds to the new wallet.

If you are running Proverb Being wallet v2.12.x:

Technically you can point Proverb Being installation to your data folder, but it is recommended to create a new installation in the default data folder. The same for your funds, it is technically possible to use the same wallet.dat file, but it is better to use the wallet created by new installation. Send your funds to this new wallet.

If you are running Proverb Being wallet v2.13.x:

You can upgrade just by replacing binaries, but backup your wallet first (wallet.dat)
